EP-DB1608无线网卡驱动安装（linux内核版本大于3.10时）

针对EP-DB1608无线网卡驱动的安装，官网只给出了linux kernel<=4.3.14的安装包。折腾了很久，决定自己上网寻找解决方案。

  • 使用lsusb命令查看系统中的usb设备，找到类似于”Bus 003 Device 002: ID 0bda:0811 Realtek Semiconductor Corp.”的条目。Realtek这个就是需要安装驱动无线网卡。ID为0bda:0811。百度或google此ID，得到其芯片组型号为RTL8812AU。
  • 根据得到的芯片组型号，搜索”linux 安装RTL8812AU”
  • 得到github地址：https://github.com/gnab/rtl8812au
  • 根据以上地址中的方法安装即可。
